Site Operation

The word to be translated is underlined. Write the translation in the text box provided. For Spanish, you can use the buttons with accents for special characters. The “Passer” button allows you to move on to the next question without answering; the answer will be given. The “Retour” button takes you back to the home page. The “Envoyer” button or the Enter key on the keyboard submit the answer. At the end of the series of questions, you'll have the option of starting the whole list again, or downloading the file containing all the errors and words you've passed.
